---
title: The Rust SDK generator
description: Use Fern's Rust SDK generator and your API definition to output a fully functional Rust SDK to your local file system, to a private repository, or to crates.io.
---

<Tip>
Coming in 2024!
</Tip>

The Rust SDK generator outputs a fully functional Rust SDK. The generator can publish the SDK to [crates.io](https://crates.io) with [Cargo](https://doc.rust-lang.org/cargo/) or to a private package manager hosted by Fern.

<CodeBlock title="generators.yml">
```yaml
- name: fernapi/fern-rust-sdk
  version: 0.0.0
```
</CodeBlock>
