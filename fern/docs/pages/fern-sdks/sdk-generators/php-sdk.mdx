---
title: The PHP SDK generator
description: Use Fern's PHP SDK generator and your API definition to output a fully functional PHP SDK (client libraries) to your local filesystem, or publicly to Packagist.
---

<Callout intent="tip">
Coming in 2024!
</Callout>

The PHP SDK generator outputs a fully functional PHP SDK. It can publish the SDK to [Packagist.org](https://Packagist.org) or to a private package manager hosted by Fern.

<CodeBlock title="generators.yml">
```yaml
- name: fernapi/fern-php-sdk
  version: 0.0.0
```
</CodeBlock>
