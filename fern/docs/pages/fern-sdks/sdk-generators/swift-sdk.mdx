---
title: The Swift SDK generator
description: Use Fern's Swift SDK generator and your API definition to output a fully functional Swift SDK to your local file system, to a private repository, or to SwiftPM.
---

<Tip>
Coming in 2024!
</Tip>

The Swift SDK generator outputs a fully functional Swift SDK. The generator can publish the SDK to [SwiftPM](https://github.com/apple/swift-package-manager), or to a private package manager hosted by Fern. You can also download the SDK to your local file system.

<CodeBlock title="generators.yml">
```yaml
- name: fernapi/fern-swift-sdk
  version: 0.0.0
```
</CodeBlock>
